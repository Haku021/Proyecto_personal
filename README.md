# People on Wheels - pag web sobre skateboard

**URL del proyecto hospedado:** https://github.com/Haku021/Proyecto_personal.git 

## Descripción:
People on Wheels es una página web para principiantes y aficionados al skateboard, donde buscamos mantenerte informados sobre los skaters más conocidos, revistas icónicas así como la experiencia de adquirir tu primer skateboard y para no perderte de nada sobre el mundo del skateboard ofrecemos una subscripción.

## Lenguajes usados:
- **HTML:** Creación de cada página
- **CSS:** Diseño y color de cada página
- **JavaScript:** Para compra de skates y productos relacionados
- **Python:** Conexión con MySQL
- **MySQL:** Almacenar pedidos y subscripción
